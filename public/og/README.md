# Open Graph Image

## Requirements

- **File:** `og-image.png`
- **Size:** 1200×630 pixels
- **Format:** PNG or JPG (NOT WebP)
- **Location:** `/public/og/og-image.png`

## Content

The OG image should represent the EditoraPDF website and include:

- **Brand/Logo:** EditoraPDF logo
- **Title/Headline:** "Edit PDF Online Free"
- **Key Features:** 
  - No Installation Required
  - No Signup Required
  - 100% Free & Private
- **Visual Elements:** PDF document icon, edit tools, modern design

## Design Guidelines

- Use high contrast for text readability
- Include brand colors (blue gradient: #3B82F6 to #2563EB)
- Keep important text in the center (safe zone: avoid outer 10% edges)
- Use professional, modern design
- Ensure text is readable at small sizes (social media thumbnails)

## Tools for Creation

1. **Canva:** https://www.canva.com/
   - Template: "Facebook Post" (1200×630)
   - Custom size: 1200×630

2. **Figma:** https://www.figma.com/
   - Create custom 1200×630 frame
   - Export as PNG

3. **Photoshop/GIMP:**
   - Create new document: 1200×630px
   - Design and export as PNG

## Testing

After creating the image, test it using:

- **Facebook Debugger:** https://developers.facebook.com/tools/debug/
- **Twitter Card Validator:** https://cards-dev.twitter.com/validator
- **LinkedIn Post Inspector:** https://www.linkedin.com/post-inspector/
